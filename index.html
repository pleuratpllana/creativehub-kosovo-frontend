Modules

The Academic Program lasts 12 months.

1 Personal Brand Building Training Students will learn::
  • How the LinkedIn algorithm works
  • Optimizing your LinkedIn profile
  • Tactics for building a professional personal brand
2 HTML Students will learn:
  • How the internet works (communication between websites and servers).
  • Getting to know the code editor.
  • Structure of HTML elements.
  • Explanation of attributes and tags and their correct use.
  • HTML-only mini web page
3 CSS Students will learn:
  • Basics of syntax
  • CSS selectors
  • CSS colors, background colors, fonts, images
  • Positioning of elements
  • Pseudo classes and pseudo elements
  • Flexbox
  • Transformations and animations
  • Responsive design - media queries
  • Using external libraries and fonts (CDN)
  • Examples from practice
  • Responsive web pages with HTML & CSS
4 Real Project 1 (Static Website) Students will produce:
  • Static web page
5 Bootstrap Students will learn:
  • Basics of syntax
  • Grid systems and grid classes
  • Typography, images, colors and background colors
  • Classes of elements and components
  • Components and their modification
  • Complete Responsive Websites with Bootstrap 5
6 Git Students will learn:
  • Learn how to set up your Git portfolio
7 JavaScript Students will learn:
  • JavaScript basics (history, console, comments, linking files defer vs. async)
  • Data Types (basic data types, constants, arithmetic operators)
  • Objects and arrays
  • Properties, methods
  • Loops
  • Functions (scope, arrow functions)
  • DOM
  • Events & Async/Await
  • AJAX
  • Classes and object-oriented programming
  • Closures & Recursions
  • Modern JavaScript
8 TypeScript Students will learn:
  • Introduction of additional JavaScript-based syntax to facilitate integration with editors to help us detect errors early
9 Real Project 2 (Dynamic Website) Outcomes:
  • A dynamic website
10 React Js Advanced Outcomes:
  • Introduction to React and Custom Components
  • JSX
  • Handling events and working with state and useState hook
  • Introduction to forms in React
  • Styling React Components
  • Component children, Conditional rendering and Debugging
  • useEffect and useReducer hooks
  • Introduction and working with Material UI
  • Styling Material UI components
  • Working with React Context and the useContext hook
  • Optimisation of React Applications
  • Client - Server communication and working with APIs
  • Creating Custom Hooks in React
  • Introduction and building multi-page applications with React Router
  • Testing React Applications
  • Securing React applications, working with Authentication
  • Building and Deploying React Applications
11 Real Project 3 Students will produce:
  • E-commerce Web Application ( in React )
12 Graduation Students will learn:
  • You will create a CV and HR portfolio
  • You will optimize your LinkedIn profile
  • You will learn how to present yourself (Elevator Pitch)
  • You will learn how to use the Requirements-Skills-Value Framework to highlight your strengths
  • You will participate in test interviews
  • You will start applying!
  • Technical interview (simulation)
